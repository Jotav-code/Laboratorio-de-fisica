Este algoritmo foi criado com o intuito de agilizar o processo de cálculo de incertezas do TIPO A, do TIPO B e do TIPO C. É importante saber como funciona o cálculo antes de usar esse algoritmo, pois um entendimento adequado das incertezas é fundamental para a obtenção de resultados precisos e confiáveis em experimentos científicos, além de garantir a aplicação correta das fórmulas e a interpretação dos dados de maneira adequada. Use com moderação!

Para melhor eficiencia, vamos usar os dados como uma matriz. Linhas X Colunas. Linhas será representada a quantidade de objetos que se quer medir, variações. Ex: medir o tempo de queda livre de uma bola em diferentes alturas, linha 1 = 60cm, linha 2 = 30 cm. De forma análoga a Coluna, vai ser a quantidade de vezes que você mediu o tempo de queda.
